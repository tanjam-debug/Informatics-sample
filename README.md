# Clinic Vaccination & Medical Record System

A web-based clinic management system built using PHP and MySQL.

## Features
- User Login System
- Patient Management
- Vaccination Records
- Medical Records
- File Upload for Attachments
- Printable Patient Summary
- Role-Based Access

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- XAMPP

## Installation Guide

1. Install XAMPP
2. Import `database/clinic_system.sql` into phpMyAdmin
3. Place folder inside `htdocs`
4. Access via:
   http://localhost/clinic-system/auth/login.php
